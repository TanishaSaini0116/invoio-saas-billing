# 💼 InvoIO — SaaS Invoice & Billing Platform

> The most elegant way to invoice clients and get paid faster.

## ✨ Features
- 🆓 First payment free — freemium model with subscription paywall
- ⚡ Real-time invoice status updates via Socket.io
- 💳 Razorpay integration — UPI, Cards, Netbanking & Wallets
- 🔐 Role-based access — Admin & Client portals (Firebase Auth)
- 📧 Email & SMS notification on payment success
- 🎊 Confetti animation on successful payment
- 🌙 Dark / Light mode toggle

## 🛠 Tech Stack
React • Node.js • Express • MongoDB • Firebase Auth • Razorpay • Socket.io • Tailwind CSS
